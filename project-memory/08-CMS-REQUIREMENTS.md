# 08 · CMS Requirements

## CMS Must Support Management Of
- Pages
- Services
- Portfolio
- Partners
- Clients
- Blog
- Careers
- Contact Information
- Site Settings
- Visual Assets

## CMS Architecture
- Custom CMS (not off-the-shelf)
- Role-based permissions

## Role-Based Access
- Admin
- Editor
- Content Manager

## Future Bilingual Support
- CMS must support bilingual content in future (English + Arabic).
- No hardcoded UI text.
