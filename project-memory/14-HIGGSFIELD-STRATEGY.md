# 14 · Higgsfield Strategy

## Position
Higgsfield is NOT the primary image generator.

## Primary Use
- Motion
- Animation
- Cinematic enhancement
- Scene transitions
- Scroll storytelling

## Images
Images may be created manually or by other tools first.

## Constraints
- All generated assets must respect project visual rules (abstract tech, no generic stock).
- Any assets containing people must respect modesty requirements for the target market.
- Product / brand identity must remain consistent.

## Reference Library
- `cinematic-landing-kit-main/` contains useful techniques but is not authoritative.
- Use only what is relevant; do not inherit its assumptions.
