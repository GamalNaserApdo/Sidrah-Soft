# SidrahSoft Project Knowledge Base

## What This Is
The official source of truth for the **SidrahSoft Website Rebuild** project.

All future analysis, planning, architecture, design, and development must align with the files in this folder.

## How to Use
1. Read files in numbered order (`00-` through `17-`).
2. Any item marked `STATUS: PENDING DECISION` requires clarification before implementation.
3. Update the decision log (`17-DECISION-LOG.md`) when major decisions are made.
4. For development rules, see `15-WINDSURF-RULES.md`.

## Non-Negotiables
- MVP scope is fixed.
- Performance > effects.
- Future academy architecture must be protected.
- No hardcoded UI text.
- Hero is cinematic, not a banner.
- Higgsfield is motion-only.
